About

Application is designed to publish your podcast in your personal application!

The application supports integration with WordPress, XML files and supports in your itunes podcast and RSS 2.0

 

 

Integration with Wordpress

Go to wordpress admin panel

Open Plugins tab -> Add new

Open tab "Upload"

Choose zip file in project folder "WP-Plugs", Image_RSS.zip and ipodcasting.zip

Upload and activate plugin

XML support

You can add your own playlist as XML file to your remote server and add a link to the file in the application.

XML sheet should be of the following structure:

<rss xmlns:content="http://purl.org/rss/1.0/modules/content/" xmlns:wfw="http://wellformedweb.org/CommentAPI/" xmlns:dc="http://purl.org/dc/elements/1.1/" xmlns:atom="http://www.w3.org/2005/Atom"xmlns:sy="http://purl.org/rss/1.0/modules/syndication/" xmlns:slash="http://purl.org/rss/1.0/modules/slash/" version="2.0">

<channel>

      <item>

              <link> http://mysite.com/podcast.mp3</link>    url mp3 link

              <title>Title podcast </title>     Title podcast

              <media:thumbnail url="http://mysite.com/image.jpg">       cover image

      </item>

</channel>

Custom Start Screen

Description start pages, you can change the file LoaderViewController.m

Open this file and change 14, 15,16 and 17 line

For change image, find image bg1@2x.png, bg2@2x.png, bg3@2x.png, bg4@2x.png open in photoeditor, change and save.

For change image logo , find image title1@2x.png, title2@2x.png, title3@2x.png, title4@2x.png open in photoeditor, change and save.
